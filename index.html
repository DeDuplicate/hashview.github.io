Hashview

A web front-end for password cracking and analytics

Check it out on GitHub

Features

  • Create predefined hashcat tasks
  • Manage jobs (set of predefined tasks)
  • Analytics and reports
  • Job queueing (no more idle time or wasted cycles)
  • retroactively update hashes previously not cracked
  • Searchable results by username, hash, or password
